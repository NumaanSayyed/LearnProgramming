In this session we will make one Calculator.So for this purpose we will use some methods and switch statement.We will define one variable as datatype char and three more variable's as datatype float.We will use char to taking user choice what user want to perform,Now we have to know what user have chose, to do this we will use next(),it is used to get the next complete token from the scanner which is in using,then we will apply charAt(0) method. It returns the character at the specified index in a string,here we have gave index 0.After that we will take user input and store it on floating variable's.Now we will start switch statement.And in the cases of switch statement we will try to match that cases with the operators that we said to user to select.Now corresponding to the user operator specific block of code will be run.Note(We have to use break keyword,after each case otherwise all cases will be run.).You can also execute the program by clicking on the Execute button. ⇓
