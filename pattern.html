In this session we will print the star pattern.First we will make two variable i and j,then we will ask to user to give input and by using nextInt() method we will take the integer input by the user.Now we will start Outer loop(It is used to printing the number of rows), and in the loop we will declare i value '0' and we'll itrate the value of 'i' whether the value of 'i' is not reach to the greater of user input value.Now we will start Inner loop(It is used to printing the number of columns), in the inner loop we will itrate the value of j whether the value of j is not reach to the equal or greater than 'i', then in the block of inner loop we will print " * ".Out of the inner loop and in the Outer loop block we will give blank space,when we will give blank space in the System.out.println(" "),So after each execution of inner loop the cursor will be at next line.Then again Outer loop will be executed and this will happen whether the condition of Outer loop is not reach to false.You can execute the Program by clicking on the Execute button. ⇓
