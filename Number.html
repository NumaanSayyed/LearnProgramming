In this article we will learn how to convert Decimal number in another Number System? 1)toBinaryString : It is used to return the string that contain the Binary equivalent of num.

2)toOctalString : It is used to return the string that contain the Octal equivalent of num.

3)toHexadecimalString : It is used to return the string that contain the Hexadecimal equivalent of num.
